# Ember Pact Demo

## How to run?

* Clone the project.

* Install `ember-cli` globally.

```
$ npm install -g ember-cli
```

* Setup `ember-client`.

```
$ cd ember-client
$ npm install 
```

* Setup `pack-mock-server`

```
$ cd ../pact-mock-server
$ npm install
``` 


Example for pact-mock-server using mocha: https://github.com/pact-foundation/pact-js/tree/master/examples/mocha
